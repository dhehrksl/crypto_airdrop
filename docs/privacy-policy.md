# 개인정보처리방침

**시행일자: 2026년 5월 21일**

"크립토 에어드랍"("앱")은 「개인정보 보호법」 제30조에 따라 이용자의 개인정보를 보호하고 권익을 보호하기 위해 다음과 같이 개인정보처리방침을 수립·공개합니다.

## 1. 개인정보의 처리 목적
- 회원 가입 및 본인 확인 (이메일 또는 Google 계정)
- 참여한 에어드랍 정보 저장 및 조회
- 푸시 알림 발송 (앱 내에서 동의한 경우)
- 서비스 개선을 위한 비식별 통계

## 2. 처리하는 개인정보 항목
| 항목 | 수집 시점 | 보유 기간 |
|---|---|---|
| 이메일 주소 | 회원가입 시 | 회원 탈퇴 시까지 |
| 사용자 이름 | 회원가입 시 | 회원 탈퇴 시까지 |
| 비밀번호(해시) | 이메일 회원가입 시 | 회원 탈퇴 시까지 |
| Google ID, 프로필 사진 URL | Google 로그인 시 | 회원 탈퇴 시까지 |
| Expo Push Token | 푸시 알림 동의 시 | 토큰 갱신 또는 탈퇴 시까지 |
| 광고 ID (AAID) | 광고 SDK가 자동 수집 | Google AdMob 정책에 따름 |
| 디바이스 정보 (OS, 모델명) | 광고/오류 분석 | 비식별화 후 통계 목적으로만 사용 |

비밀번호는 bcrypt 단방향 해시로만 저장되며, 평문 형태로 보관하지 않습니다.

## 3. 개인정보 처리 및 보유 기간
원칙적으로 개인정보 수집·이용 목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다. 회원 탈퇴 요청 시 즉시 삭제하며, 다음과 같은 법령상 의무 보존 정보는 명시된 기간 동안 보관합니다.
- 「전자상거래 등에서의 소비자보호에 관한 법률」에 따른 기록: 해당 없음 (앱은 거래·결제 기능을 제공하지 않음)

## 4. 개인정보의 제3자 제공
앱은 이용자의 개인정보를 외부에 제공하지 않습니다. 다음의 경우는 제외합니다.
- 이용자가 사전에 동의한 경우
- 법령에 따라 수사기관 또는 감독기관의 적법한 요청이 있는 경우

## 5. 개인정보 처리의 위탁
| 수탁사 | 위탁 업무 | 위탁 범위 |
|---|---|---|
| Expo (Expo, Inc.) | 푸시 알림 전송 | Expo Push Token |
| Google LLC | Google 로그인 인증, 광고 표시 (AdMob) | 인증 토큰, 광고 ID |
| Google LLC (Gemini API) | 뉴스/에어드랍 텍스트 분류·번역 | 공개 RSS 기사 텍스트 (개인정보 미포함) |

## 6. 정보주체의 권리·의무
이용자는 언제든지 다음 권리를 행사할 수 있습니다.
- 개인정보 열람 요구 (앱 내 "내 정보" 화면)
- 개인정보 정정·삭제 요구 (앱 내 "내 정보 > 회원 탈퇴")
- 개인정보 처리 정지 요구


## 7. 개인정보의 안전성 확보 조치
- 비밀번호 단방향 해시화 저장 (bcrypt)
- HTTPS를 통한 통신 암호화
- 접근 권한 최소화

## 8. 만 14세 미만 아동의 개인정보
앱은 만 14세 미만 아동의 회원가입을 받지 않으며, 광고 노출 설정은 `tagForChildDirectedTreatment=false`로 13세 이상 일반 사용자를 대상으로 합니다.

## 9. 광고 식별자
이 앱은 Google AdMob을 사용하여 광고를 표시하며, 광고 ID(AAID)는 광고 빈도·관련성 제어 목적으로만 사용됩니다. Android 설정에서 광고 ID 재설정·삭제가 가능합니다.
